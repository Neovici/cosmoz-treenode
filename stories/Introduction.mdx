# Cosmoz Treenode

`cosmoz-treenode` is a web component that displays a node in a tree data structure. It's part of the Cosmoz suite of components.

## Features

- Display a node's path in a tree structure
- Customize the display with various options:
  - Hide nodes from the root
  - Show a maximum number of nodes
  - Customize the separator between path parts
  - Add ellipsis for shortened paths
  - Provide fallback text when a path cannot be found
  - Control text wrapping

## Usage

```html
<cosmoz-treenode
  key-property="id"
  key-value="your-node-id"
  search-property="name"
  path-string-separator=" / "
></cosmoz-treenode>
```
